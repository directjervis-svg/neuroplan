# Padrões de Design Analisados - NeuroPlan v2

## Layout com Abas Dinâmicas (Accordion)
- Header preto com texto branco
- Estado ativo: cor destaque (#FF6B35 laranja)
- Ícone de seta que rotaciona 180° quando ativo
- Meta informações (tempo, status) no header
- Animação suave de max-height para expansão
- Indicadores de fase circulares no topo

## Padrão de Exportação PDF
- Paleta corporativa: navy-primary (#0F172A), navy-secondary (#1E293B)
- Accent colors: green-success (#059669), amber-warning (#D97706), red-danger (#DC2626), blue-accent (#2563EB)
- Dimensões A4: 210mm x 297mm
- Padding: 20mm
- Header stripe com gradiente
- Footer com data e paginação
- SWOT Matrix com grid 2x2
- Gauges circulares para métricas
- Cards com border-radius: 4px

## Calendário Mensal
- Grid 7 colunas (DOM-SÁB)
- Dias fora do mês em cinza claro
- Eventos com cards coloridos por categoria:
  - PRODUTO: borda esquerda cinza
  - LANÇAMENTO: fundo preto, texto branco
  - MARCO: fundo preto, texto branco
  - VALIDAÇÃO: borda esquerda cinza
- Título do mês + ano no topo
- Nome do projeto abaixo
- Data de geração no footer

## 5 Fases de Gestão de Projetos (PMBOK)
1. INICIAÇÃO - Definir escopo, stakeholders, viabilidade
2. PLANEJAMENTO - Cronograma, recursos, riscos, orçamento
3. EXECUÇÃO - Implementar, coordenar, comunicar
4. MONITORAMENTO - Controlar, medir, ajustar
5. ENCERRAMENTO - Documentar, avaliar, celebrar
